Comic Subscription Web Application

Project Overview

Developed a web-based application that allows users to subscribe to Marvel comic series and receive notifications when new issues are released, using SMS and email notifications via the Twilio and SendGrid APIs.

GitHub Repository: View Project on GitHub

Core Features

  • Marvel Character Search - Search for your favorite Marvel characters, and the app retrieves a list of comic series related to them using the Marvel API.
  • Subscription System - Users can subscribe to any comic series and receive real-time notifications whenever new issues are available.
  • Email and SMS Notifications - Automated notifications are sent using SendGrid for email and Twilio for SMS when new chapters are released.

Technologies Used

  • Frontend: HTML, CSS, JavaScript
  • Backend: Node.js, Express.js
  • Database: MongoDB (via Mongoose) for storing user subscriptions
  • APIs: Marvel API, Twilio API, SendGrid API for fetching comic data and sending notifications

RESTful API

  • GET /api/comics/:character: Fetches comic series data for a specific character from the Marvel API.
  • POST /api/subscribe: Subscribes a user to a selected comic series using their email and phone number.
  • DELETE /api/subscribe/:id: Unsubscribes a user from a comic series based on the unique ID.

User Interface

  • Responsive design using HTML, CSS, and JavaScript.
  • Features input validation, error handling for comic searches, and dynamic displays of comic series and subscription status.
