Predicting NFL MVPs

This project is a comprehensive analysis of over 30 NFL seasons to predict future MVP candidates using statistical models. Key variables like passing yards, rushing touchdowns, and player fatigue were examined to draw meaningful insights.

Project Overview

This project uses techniques such as Linear Regression and MANOVA to predict future MVP candidates by analyzing player statistics like passing yards, rushing touchdowns, and more.

Technologies Used

  • Programming Language: R
  • Analysis Tools: RStudio, MANOVA, Linear Regression
  • Data: NFL Statistics from the past 30 seasons

Conclusion

The project successfully identified key patterns and trends among MVP candidates, using regression models and decision trees to provide predictions for future NFL MVPs.
